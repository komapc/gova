GPS kaj Alteco

  • GPS-alteco uzas WGS84-elipsoidan referencon — ne ĉiam identan al marnivelo
  • Kiam rete, Open-Elevation API provizas MSL-korektitan altecon
  • Alteca precizeco dependas de GPS-signalkvalito (tipe ±3–15 m)
  • Loknomoj devenas de OpenStreetMap per Nominatim

Privateco

  • GPS-koordinatoj uzatas nur por alteco-montrado
  • Neniaj datumoj estas senditaj al ni — ĉio okazas sur via aparato
  • Reverse geocoding (loknomoj) sendas koordinatojn al OpenStreetMap — nur kiam rete
  • MSL-korekcio sendas koordinatojn al Open-Elevation API — nur kiam rete
  • La lasta konata alteco estas konservita loke sur via aparato
